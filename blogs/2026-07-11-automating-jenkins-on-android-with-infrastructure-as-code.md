From Manual to Automated In March 2023, I explored running Jenkins on Android devices using Termux, demonstrating that it’s technically possible to transform aging smartphones into CI/CD infrastructure. The manual setup worked, but it required 2-3 hours of configuration and was error-prone. Fast-forward to 2025: I’ve automated the entire process using Infrastructure as Code principles.
