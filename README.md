# Jenkins Pipeline (jenkins-pipeline)

Jenkins Pipeline is a suite of plugins for the Jenkins automation server that supports implementing and integrating continuous delivery pipelines as code. It provides an extensible domain-specific language (DSL) with two syntax styles - Declarative and Scripted - and core building blocks (agents, stages, steps) for modeling simple-to-complex CI/CD workflows that are versioned with application source code via a Jenkinsfile.

## APIs

Jenkins Pipeline is a configuration-as-code DSL rather than a callable HTTP API. Programmatic interaction with pipelines and builds is performed through the Jenkins Remote Access API, which is indexed in the companion `jenkins` repository.
